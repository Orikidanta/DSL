#  智能电商客服机器人（基于 LLM + 状态机）最后版本

本项目是一个轻量级、可扩展的电商客服对话系统，结合大语言模型（LLM）与规则驱动的状态机，支持**物流查询、投诉受理、退款申请**三大核心场景，并能智能处理用户“一句话”完整请求（如“查物流，订单号123456”）。

>  **无需硬编码意图识别**｜**上下文自动管理**｜**真实 LLM 驱动**｜**易于测试与扩展**

# 支持自定义扩展
添加新场景只需在qwen_client.py中的system_prompt加入对应的scence和status,并于rules.txt中添加新规则即可

# 启动对话
打开文件，运行main.py即可。另有自动脚本测试test_bot.py
注：因为脚本测试有缺陷所以最后一个显示错误，对比实际输出其实是对的
